Step 1: By Hand
* 0/15/1999 to 10/25/1999: Same month and year. 25 − 15 = 10 days.
* 10/23/1999 to 12/1/1999:
Remaining days in October after the 23rd: 31 − 23 = 8 days (reaches Oct 31).
Full November: 30 days.
December 1: +1 day.
Total: 8 + 30 + 1 = 39 days.
* 10/21/1999 to 3/4/2004:
Remaining in 1999 (Oct 21 → Dec 31): Oct (31-21=10 days to Oct 31) + Nov 30 + Dec 31 = 71 days.
Full years 2000–2003: 2000 (leap, 366) + 2001 (365) + 2002 (365) + 2003 (365) = 366 + 3×365 = 1461 days.
2004 (leap year) up to March 4: Jan 31 + Feb 29 + Mar 4 = 64 days (but adjust for exclusive start: from Jan 1 it is the 64th day of the year, so +63 days from Jan 1).
Grand total: 71 + 1461 + 64 − 1 (off-by-one adjustment) = 1596 days.
Step 2: Approach
1. Check if a year is a leap year using the Gregorian rule: divisible by 4, but not by 100 unless also divisible by 400.
2. Add up all complete years from the base year to the year before the target date (adding 365 or 366 days per year).
3. For the target year, add the days from all completed months before the target month, using a days-per-month lookup (with February adjusted to 29 in leap years).
4. Finally, add (day – 1) so the count is exclusive of the starting date.

FUNCTION date_to_serial(year, month, day):
    total_days = 0
    
    // Add days from all complete years since base year (e.g., 1900)
    FOR y = 1900 TO year - 1:
        IF is_leap_year(y) THEN
            total_days = total_days + 366
        ELSE
            total_days = total_days + 365
    
    // Add days from complete months in the current year
    FOR m = 1 TO month - 1:
        total_days = total_days + days_in_month(m)
    
    // Add leap day if this year is a leap year and we have passed February
    IF month > 2 AND is_leap_year(year) THEN
        total_days = total_days + 1
    
    // Add the day of the month (subtract 1 for exclusive start date)
    total_days = total_days + (day - 1)
    
    RETURN total_days


FUNCTION days_between(y1, m1, d1, y2, m2, d2):
    serial1 = date_to_serial(y1, m1, d1)
    serial2 = date_to_serial(y2, m2, d2)
    RETURN serial2 - serial1
